# LIN — pkboie 個人工作記錄

這個資料夾是 **pkboie (LIN)** 個人用來記錄開發過程的地方，不是遊戲資源。
不會被 Cocos Creator 當成 asset 載入（因為不在 `assets/` 底下）。

## 用途

- 記錄每次開發做了什麼、為什麼這樣做
- 暫存還沒整理好的筆記、設計草稿
- 個人的 TODO、想法、實驗紀錄

## 規則

- ✅ 只有 LIN（pkboie）自己會編輯這個資料夾
- ✅ 這裡的檔案不影響遊戲執行
- ❌ 不要把遊戲資源（prefab、png、ts、scene）放這裡 —— 那些要放 `assets/` 底下並遵守 [TEAM_GUIDE.md](../TEAM_GUIDE.md)
- ❌ 組員不要動這個資料夾

## 建議檔案

- `notes/` — 開發筆記
- `todo.md` — 個人待辦
- `log.md` — 每日進度日誌
